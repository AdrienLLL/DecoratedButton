Implémentation du Pattern décorateur sur un JButton : 

Trois décorateurs utilisés : - HideOnClick/- AlertOnClick/ - GrossBorder

Une classe Main

HideOnClick permet de cacher un bouton lors du clic sur celui-ci

AlertOnClick permet de faire apparaître une fenêtre d'alerte lors du clic sur le bouton

GrossBorder permet de grossir le contour du bouton lors du clic sur celui-ci
